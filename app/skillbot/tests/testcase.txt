; Skillbot Debugger Transform Test Cases
; Format:
;   [testcaseNN] description
;   INPUT:
;   <SKILL source code>
;   OUTPUT:
;   <expected transformed code>
;
; Lines starting with ';' are comments (ignored by test runner).
; INPUT: and OUTPUT: markers must be on their own line.
; Each test case ends at the next [testcaseNN] or EOF.


[testcase01] Basic procedure with simple statements
INPUT:

    a = 10
    printf("a => %L" a)

    ))

    ))
